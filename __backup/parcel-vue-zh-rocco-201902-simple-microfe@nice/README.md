## Project setup

### Compiles and hot-reloads for development

```bash
# install dependencies
yarn ins

# development serve

# dev entry-app
cd entry-app
yarn serve

# open a new shell, dev sub-app
cd sub-app-ts
yarn serve

# open a new shell, dev sub-app
cd sub-app-js
yarn serve

# open: http://localhost:7300/
```

### Compiles and minifies for production

```bash
# production build
yarn build

# production build with serve
yarn build:serve

# open: http://localhost:3000/
```
